1‑Month Fast‑Track — Java Spring Boot + Angular Full Stack

Intensive, hands‑on syllabus with a mini project — built for rapid skill gains and a portfolio-ready outcome.

Duration: 4 weeks • Format: Live / Recorded
W1

Week 1 — Core Java & Spring Boot Foundations

A fast Java refresher followed by Spring Boot basics to build REST APIs and connect to a relational database.

  • OOP, Collections, Streams & Lambdas
  • Spring Boot annotations, Controllers, Services, Repositories
  • application.yml, REST endpoints, DB connection (MySQL/Postgres)
W2

Week 2 — Backend Deep Dive

Design robust REST APIs with validation, error handling, and basic security. Learn testing essentials.

  • CRUD patterns, DTOs, Model mapping
  • Validation (@Valid), Global Exception Handling
  • Spring Security basics, JWT overview, Unit & Integration tests
W3

Week 3 — Angular Fundamentals

Build responsive, component-driven UIs that consume your Spring APIs.

  • Angular CLI, Components, Modules, Data binding
  • HttpClient, Observables (RxJS), Routing, Reactive Forms
  • Integration patterns and error handling
W4

Week 4 — Integration & Mini Project

Finish a full-stack mini project, deploy locally or in Docker, and prepare a GitHub-ready portfolio.

  • Connect Angular frontend to Spring Boot backend
  • CRUD flows, authentication (JWT or in-memory), E2E checks
  • Deploy backend (local/Docker), host frontend (static hosting)
Mini Project (Suggested)
Task Management System — Users can create, update, view & delete tasks. JWT-based auth (optional).
Stack: Java, Spring Boot, MySQL/Postgres, Angular, Docker (optional)
